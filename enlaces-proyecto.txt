  NOTA SOBRE LA “PÁGINA” DEL SITIO
--------------------------------------------------------------------------------
  Este proyecto es un sitio estático (HTML/CSS/JS) + API en carpeta backend.

  • Vista previa en GitHub (archivos, sin ejecutar la API):
    Abre el repositorio y navega a index.html, login.html, etc.

  • Para usar login/registro con fetch: levanta la API (carpeta backend,
    "npm run dev") y abre el frontend con un servidor local, por ejemplo
    Live Server en VS Code (no uses solo file:// si hay problemas de CORS).

  • Si en el futuro activas GitHub Pages en el repo, la URL suele ser del tipo:
    https://aixav1603-png.github.io/unidad-1-de-html/
    (solo funciona cuando está habilitado en Ajustes → Pages del repositorio.)
